Add options to hide Gravity Forms field labels and and sub-labels.

== Description ==

Adds a new Field Label Visibility setting and an option for hidden sub-labels under the Appearance tab in the Gravity Forms field settings.

Requirements:

* [Gravity Forms v1.9+](https://www.gravityforms.com/) is required to make use of this plugin.

1. Field Label Visibility and Hidden Sub-labels

== Installation ==

* Make sure that [Gravity Forms v1.9+](https://www.gravityforms.com/) is installed and activated.
* Download and install using the built in WordPress plugin installer.
* Activate in the "Plugins" area of your admin by clicking the "Activate" link.
* No further setup or configuration is necessary.
